# Security Policy

## Supported Versions

Currently, this repository is for educational purposes and we do not maintain long-term support branches. However, we strive to keep our dependencies, tools, and examples as secure and up to date as possible.

## Reporting a Vulnerability

If you discover a security vulnerability within this repository, please do not open a public issue. Instead, please send a direct message to the repository owner or open a private vulnerability report via GitHub.

We will endeavor to respond and fix the issue as quickly as possible. Your efforts to responsibly disclose security issues are highly appreciated.
