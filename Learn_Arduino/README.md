# 🚀 Learn Arduino: The God-Tier Beginner's Guide

Welcome to **Learn Arduino**, your ultimate Zero-to-Hero guide to mastering embedded systems!

If you're here, you are probably getting ready to tackle the massive **100 Days of Arduino** challenge. But before you dive into writing non-blocking code, sensor fusion, and autonomous robotics, you need a solid foundation. You need to know your tools.

This guide is designed to take you by the hand and explain **everything** you need to know about Arduino, starting from absolute zero.

## 🛠️ How to use this guide

Read through these chapters sequentially. They are designed to build upon each other.
Whenever you see a code block, try to type it out yourself in the Arduino IDE instead of copy-pasting. Muscle memory is incredibly important in programming!

> [!TIP]
> **No hardware? No problem!**
> If you don't have your Arduino kit yet, you can still follow along! Use an online simulator like [Wokwi](https://wokwi.com/) or [Tinkercad Circuits](https://www.tinkercad.com/circuits) to build circuits and write code right in your browser.

Let's get started. Click on [Chapter 1](./Chapter_01_What_Is_Arduino.md) and take your first step into the world of embedded systems!
