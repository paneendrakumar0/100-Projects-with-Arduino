# Learn Arduino

A beginner-friendly foundation for the 100 Days of Arduino curriculum.

Read this guide before Day 1 if you are new to Arduino, electronics, breadboards, or embedded C++. It introduces the tools, vocabulary, wiring habits, and programming patterns you need before the project sequence gets serious.

## How to Use This Guide

Read the chapters in order the first time through. Type short code examples into the Arduino IDE yourself when possible; it builds the muscle memory you will need during the project days.

No hardware yet? Use [Wokwi](https://wokwi.com/) or [Tinkercad Circuits](https://www.tinkercad.com/circuits) to simulate simple circuits in your browser.

When you finish Chapter 16, continue to [Day 1: The Non-Blocking Blink](../Day_01_Millis_Blink).
